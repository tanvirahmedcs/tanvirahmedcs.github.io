2. **Upload files**:

   - Drag and drop all files from this folder
   - Commit changes

3. **Enable GitHub Pages**:
   - Go to **Settings** â†’ **Pages** (left sidebar)
   - Under "Build and deployment":
     - Source: **Deploy from a branch**
     - Branch: **main** (or master)
     - Folder: **/(root)**
   - Click **Save**
   - Wait 1-2 minutes for deployment

4. **Your site is live at**:

## ğŸ“ˆ Submit to Search Engines

After deploying, submit your sitemap to:

1. **Google Search Console**:
   - Go to [search.google.com/search-console](https://search.google.com/search-console)
   - Add your site URL
   - Verify ownership (DNS or HTML file)
   - Submit sitemap: `https://yoursite.com/sitemap.xml`

2. **Bing Webmaster Tools**:
   - Go to [bing.com/webmaster](https://www.bing.com/webmaster)
   - Add your site
   - Submit sitemap

3. **Google Analytics** (optional):
   - Create account at [analytics.google.com](https://analytics.google.com)
   - Add tracking code to `<head>` section
